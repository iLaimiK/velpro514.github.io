# 关于语法

我认为语法是个很玄学的东西，在你没了解某个语言的体系、语言的思维习惯之前，可能连该语言中那些简单的语法都觉得很难。如果用死记硬背的方法学习语法，最后只能是事倍功半，收效甚微。这里列出只为整理笔记，顺便加深记忆。

语法意义解释例句
A は B です… 是… / … 在…「は」提示助词，表强调。它在做助词时读音为「わ」。 「です」判断助动词，表 “是” 或 “在”。1.私(わたし) は学生(がくせい) です。2.食堂(しょくどう) はデパートの七階(ななかい) です。
A は B ではありません… 不是…「ではありません」是「です」的否定形式。私は学生ではありません。
A は B ですか… 是… 吗？助词「か」接在句尾表示疑问，且日语问句在句尾不使用 “？”。あなたは学生ですか。
A は B ですか，C ですか… 是 / 在… 还是…？答案有多种可能，询问其中一种时，可重复使用谓语「ですか」今日(きょう)は水曜日(すいようび)ですか，木曜日(もくようび)ですか。
A はどこですか… 在哪里用于询问存在的场所，A 一般为地点名词，也可为物品あなたのかばんはどこですか。
A はいくらですか… 多少钱询问价钱时，用「いくら」その服(ふく) はいくらですか。
A に B があります／います某处有某人 / 某物表事物存在，「あります」用于植物、物品等不具有意志的事物，「います」用于具有意志的人、动物或昆虫庭(にわ) に何(なに) がありますか。
A は B にあります／います某人 / 某物在某处表事物位置犬(いぬ) はどこにいますか。
上(うえ)/下(した)/前(まえ)/後(うし) ろ/隣(となり)/中(なか)/外(そと)上面 / 下面 / 前面 / 后面 / 旁边 / 中间；里面 / 外面表示具体位置时，用 “名词 + の + 前面一系列方位词会社(かいしゃ) の隣(となり) に花屋(はなや) があります。
A も B です… 也是…助词「も」基本相当于汉语的 “也”私(わたし) も中国人(ちゅうごくじん) です。
(疑问词)＋も＋(动词否定形式)什么都没有表示全面否定教室(きょうしつ) に誰(だれ) もいません。
A と B… 和…助词「と」加在两个名词之间表示并列居間(いま) にテレビとビデオがあります。
A の B… 的…助词「の」连接名词和名词，表示所属ヂュポンさんは大学(だいがく) の先生(せんせい) です。
そうです / そうじゃありません /違(ちが) います是的 / 不是 / 说错了（不太礼貌）回答疑问时用A：これはあなたのですか。 B：いいえ、そうじゃありません。 B：いいえ、違いますよ。
これ / それ / あれ / どれ表东西这 (近)/ 那 (远)/ 那 (更远)/ 哪个 (表疑问)，具体含义自行查询それは何(なん) ですか。
この / その / あの / どの表东西和人修饰名词，这个 (近)/ 那个 (远)/ 那个 (更远)/ 哪个 (表疑问)このカメラはスミスさんのです。
ここ / そこ / あそこ / どこ表场所这里 (近)/ 那里 (远)/ 那里 (更远)/ 哪里 (表疑问)そこは図書館(としょかん) です。
こちら / そちら / あちら / どちら表场所、方向、人物指代范围比上一个更大，这边 (近)/ 那边 (远)/ 那边 (更远)/ 哪边 (表疑问)こちらは木村(きむら) さんです。
何：なん / なに什么1. ただな行前读なん；2. 用于疑问词 / 量词时读なん；3. 其他读なに1.何(なん) と言(い) いましたか。2.何歳(なんさい) ですか。3.明日何(あしたなに) をしますか。
ね语气词1. 表确认；2. 表感叹；3. 期待对方同意；4. 表叮咛嘱咐1.宋(そう) さんですね。2.今日(きょう) も暑(あつ) いですね。3.明日(あした) も来(き) ますね。4.明日遅刻(あしたちこく) しないでね。
今(いま)～時(じ)～分(ふん)です现在几点几分表示现在的时间，三十分(さんじゅっぶん)可以用半(はん)替代，表时间的词前面还可以加上午前(ごぜん)或午後(ごご)，询问时间用何時(なんじ)今午前八時半(はちじはん)です。
A に B表动作发生时间，A 为时间，B 为动词表示动作发生的时间时，要在具体时间词语后面加上助词「に」。学校(がっこうは) は八時半(ちじはん) に始(はじ) まります。
A から B まで C什么时候到什么时候，A 和 B 皆为时间，C 为动词表示某动作发生在某个期间，「から」和「まで」也可分别单独使用1.森(もり) さんは月曜日(げつようび) から水曜日(すいようび) まで休(やす) みました。2.私(わたし) は九時(くじ) から働(はたら) きます。3.森(もり) さんは二時(にじ) まで勉強(べんきょう) します。
いつ A ますか什么时候…？ A 为动词询问某动作或事态进行的时间試験(しけん) はいつ始(はじ) まりますか。

# 关于时态变化

一般现在时过去时否定过去否定
ますましたませんませんですた
いかったくないです / くありませんくなかったです / くありませんでした
ですでしたでは　ありませんでは　ありませんでした
  • Examples：
  1. ます

私は電車(でんしゃ)で東京から横浜まで行きます。（我从东京坐电车去横滨。）

森さんは昨日(きのう)9 時から 6 時まで働(はたら) きました。（森先生昨天从 9 点工作到 6 点。）

私は今日(きょう)タバコを吸(す) いません。（我今天不吸烟。）

森さんは今朝何(けさなに)も食(た) べませんでした。（森先生今天早上什么都没吃）

  1. い

なにを読(よ)んでいるんだい。（你看什么哪？）（这里作语气词）

去年(きょねん)の今頃(いまごろ)は雨(あめ)が多(おお)かった。（去年这个时候雨很多。）

初夏(しょか) はまだ暑(あつ) くないです。（初夏还不是很热。）
この種(たね) の人間(にんげん) も少(すく) なくありません。（这类人也不少。）

私は中古車(ちゅうこしゃ)を買(か)いましたがあなたが思(おも)っていたほど高(たか)くなかったです。（他预料到了这辆车的价格，但是并没有那么高。）
彼(かれ)はこの車(くるま)の値段(ねだん)を予想(よそう)しましたがそれほど高(たか)くはありませんでした。（我买了一辆二手车，但没有你想象的那么贵。）

  1. です

彼女(かのじょ) が次期議長(じきぎちょう) に推薦(すいせん) された人(ひと) です。（她是被推荐为下一任主席的人。）

昨日(きのう) はいい天気(てんき) でした。(昨天是好天气。)

この病院(びょういん) の受付(うけつけ) は午前中(ごぜんちゅう) ですが、急病(、きゅうびょう) の場合(ばあい) はこの限(かぎ) りではありません。（本医院挂号是上午，但急诊不在此限。）

先週(せんしゅう，)，この駅(えき) は静(しず) かではありませんでした。（上周这个车站并不安静。）
